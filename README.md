# PETCARE 
เว็บไซต์จำลองของ **PETCARE ** สำหรับ **Assignment #1**  
เป็นเว็บไซต์บริษัทPETCARE บริษัทรับฝากดูแลสัตว์เลี้ยง

## รายละเอียดข้อมูล
เว็บไซต์ประกอบด้วย **4 หน้าหลัก** ดังนี้ :

1. หน้าแรก (Home)
2. เกี่ยวกับเรา (About)
3. บริการ (Services)
4. ติดต่อเรา (Contact)

## โครงสร้างไฟล์
